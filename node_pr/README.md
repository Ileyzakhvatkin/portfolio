## Как локально развернуть проект
1. FronEnd проекта был создан на Node ``v.14`` проверить соответствие версии на локальном ПК
2. Установить зависимости npm i
3. Из файла .env-sample создать файл .env, в котором указать данные своей базы типа PostgreSQL
4. Запустить сборку front-end приложение npm run build
5. Запустить сборку в режиме DEV back-end приложения npm run dev - команда так же создаст структуру базы данных
6. Запустить сборку в режиме PROD back-end приложения npm run start

## В проекте реализованы следующие БОНУСНЫЕ возможности
1. Для нового пользователя автоматически создаётся демо-заметка
2. Работает поиск по заголовкам заметок
3. Пользователь может скачать PDF заметки
